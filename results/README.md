# Archived Results

This directory contains lightweight artifacts for reproducing the reported tables and figures.

| File | Description |
|---|---|
| `cv_results.csv` | Per-fold metrics from the 10-fold paper-aligned CV experiment. |
| `ablation_results.csv` | Mean and standard deviation for core ablation variants. |
| `oof_roc.png` | Out-of-fold ROC curve. |
| `oof_pr.png` | Out-of-fold precision-recall curve. |

Large intermediate files, such as balanced training data, checkpoints, and logs, are kept out of version control.
